Component-based Angular JS (1.5+) with Web API 2 (via CORS), EF 6 Migrations

ShreeVenkat.co.uk is an attempt to learn and showcase the behind-the-scenes aspect of new frameworks & features of .NET & web development, both server & client platforms. Since I've put out the source code of this website on GitHub in 2015, I've created (& deleted) numerous GitHub repositories and accounts - trying to keep only the relevant & newer frameworks of application development.

This is currently version 2 of my website that uses Angular JS 1.x (over shell MVC 5) on client-side and Web API 2 via CORS (Api.ShreeVenkat.co.uk), Entity Framework 6 Migrations & SQL Server 2012 on the server-side. Blog posts & dropbox-simulated files are stored in the database and a CMS is provided to manage the data. While you can login and browse through data, POST permissions are Admin-Authorised. The CMS itself is a generated code from Visual Studio templates with CRUD operations using Entity Framework.

I've implemented my own Blog and it is retained at version 1 to demonstrate KnockoutJS usage, and hence you can see this SPA loading Blog (& CMS) pages as they're hosted under different subdomains. I've implemented file storage solution to emulate Dropbox or Google Drive, JS compiler to emulate CodePen or JS Fiddle, and retail shopping portals to demonstrate varied development experience.

This website is built/packaged using npm (for build libs / devDependencies) & bower (for browser-side framework libs) package managers, Grunt and (optionally) Gulp JS task runners for JSHint-ing and minifying & bundling static files, CDN usage (for heavy libs) with fall backs, error handling on client-side (Angular) & server-side (MVC), authentication & authorization using ASP.NET Identity & OAuth, Entity Framework 6 Migrations for database change deployments, and much more...

A full-on Continuous Delivery is followed as my development process using Git and Jenkins. Take a look at my repositories @ https://github.com/shree-venkat.
